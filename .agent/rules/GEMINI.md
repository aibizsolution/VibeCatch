# 언어 관련 기본 규칙 (강제)

- 에이전트가 생성하는 **모든 답변, 설명, 보고, 상태 업데이트는 반드시 한국어**로 한다.
- 사용자에게 되묻거나 질문할 때도 **항상 한국어로 질문**한다.
- 코드 내부의 주석을 작성하거나 변경할 때도 **항상 한국어를 사용**한다.
- 영어 또는 다른 언어로 답변할 필요가 있을 경우에도, **먼저 한국어로 설명한 뒤**, 필요한 부분만 영어 단어를 보조적으로 사용할 수 있다.
- 사용자의 요청에 영어가 포함되어 있어도, **출력 결과와 대화는 한국어를 우선**으로 한다.

# 주석 처리 규칙

- 기존 한국어 주석은 언어와 의미를 유지하고, 꼭 필요한 경우에만 최소한으로 수정한다.
- 한국어 주석을 영어/중국어/일본어/한자 등 다른 언어로 바꾸지 않는다.
- 새 주석이 필요할 때는 반드시 **한국어로만 작성한다**.
- 코드 리팩터링 시, 기존 주석은 가급적 수정 없이 유지한다.

# 일반 행동 지침

- 가능하면 단계적으로, 한국어로 진행 상황을 간단히 보고한다.
- 사용자가 명시적으로 영어를 요구하지 않는 한, **대화·작업·지시·해설 모든 출력은 한국어가 기본값**이다.
- 모든 파일은 **처음부터 UTF-8로 읽는다**. (판단 없이 강제)
- 작업 시작 전에 **루트의 README.md를 UTF-8로 먼저 읽고** 그 규칙과 구조를 따른다.
